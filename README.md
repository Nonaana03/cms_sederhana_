# CMS Sederhana (MVC)

Ini adalah Content Management System (CMS) sederhana menggunakan pola arsitektur MVC (Model-View-Controller) dengan PHP.

## Struktur Folder
```
cms_sederhana_/
  ├── app/
  │   ├── controllers/
  │   ├── models/
  │   └── views/
  ├── public/
  │   └── index.php
  ├── core/
  │   ├── App.php
  │   ├── Controller.php
  │   └── Model.php
  ├── .htaccess
  └── README.md
```

## Fitur
- CRUD Artikel
- Routing sederhana
- Struktur MVC
